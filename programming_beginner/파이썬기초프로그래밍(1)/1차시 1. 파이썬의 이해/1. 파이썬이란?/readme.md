# Python
## 폭넓은 사용자층을 가진 프로그래밍 언어
- 빠른 학습속도
- 명령의 수행 결과를 빠르게 확인 가능
- 확장 기능 지원
- 다양한 플랫폼에서 사용 가능 
- 성능에 영향을 미치는 부분은 C언어등으로 모듈화 
- 독립적이며, 인터프리터 방식의 객체지향이고,
- 동적인 그리고 대화형 성격을 가진 프로그래밍 언어
  
## 역사
- 1980 년대 말 고안
- 1991 2. 20 귀도 반 로섬 발표 

## 유니코드
- 각 나라별 언어를 모두 표현하기 위해 만든 통합 코드체계
- 16비트
- 최대 65,536자 표현 가능
- 파이썬에서 사용되는 문자열 = 유니코드 문자열
